# RAG - 检索增强生成

让 AI 基于你的文档回答问题，而不是编造答案。

## 特性

- 文档导入（Markdown / PDF / 网页）
- 向量检索 + 语义匹配
- 回答带引用（citation）
- 可替换检索后端

## 流程

```
文档 → 切块 → 向量化 → 存入向量库
用户提问 → 向量化 → 检索相似块 → 注入上下文 → LLM 回答（带引用）
```

## 可用实现

| Adapter | 说明 | 配置值 |
|---------|------|--------|
| LlamaIndex | 功能完整的 RAG 框架 | `llamaindex` |

## 演进路径

| 阶段 | 能力 | 候选方案 |
|------|------|---------|
| V0 | 不做 | — |
| V1 | 基础向量检索，文档导入，带引用回答 | LlamaIndex + Qdrant |
| V2 | 混合检索（向量+关键词）、Rerank、多文档 | Cohere Rerank, BM25 |
| V3 | GraphRAG、Agentic RAG、多模态文档 | LightRAG, 知识图谱 |

## 可扩展能力

- Chunking 策略可插拔（固定大小 / 语义分割 / 文档结构感知）
- Embedding 模型可替换
- 检索策略可组合（向量 / 关键词 / 图 / 混合）
- 索引更新策略（实时 / 批量 / 增量）
- Rerank 模型可插拔

## 扩展方式

实现 `RetrievalPort` 接口 + 注册即可。
